## Upgrading to the Latest Version

Upgrade Mosambee effortlessly to access the newest features and security patches. Follow these steps to update your integration.

<Callout kind="info">
  Backup your configuration before upgrading. Test in a staging environment first.
</Callout>

<Steps>
  <Step title="Check Current Version" icon="package">
    Run the version check command in your terminal.
  </Step>
  <Step title="Install Update" icon="download">
    Use your preferred package manager.

    <CodeGroup tabs="npm,yarn">
````bash
npm install mosambee@latest
````
````bash
yarn add mosambee@latest
````
    </CodeGroup>
  </Step>
  <Step title="Restart Service" icon="refresh-cw">
    Restart your Mosambee instance and verify endpoints like `https://api.example.com/metrics`.
  </Step>
</Steps>
