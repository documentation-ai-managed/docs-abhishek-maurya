## Prerequisites

Before starting, ensure you have:

- A valid email address for account creation
- Company details ready (name, address, size)
- Employee information (names, emails, roles)

<Callout kind="info">
  Mosambee requires a stable internet connection for dashboard access. Use modern browsers like Chrome or Firefox for the best experience.
</Callout>

## Create Your Account

Sign up for Mosambee to access the workforce management dashboard.

<Steps>
  <Step title="Visit Signup Page" icon="globe">
    Navigate to `https://app.mosambee.com/signup`.
  </Step>
  <Step title="Enter Details" icon="user-plus">
    Provide your company name, email, and password. Verify your email via the confirmation link sent to your inbox.
  </Step>
  <Step title="Complete Onboarding" icon="check-circle">
    Answer basic questions about your workforce size and industry. This helps customize your dashboard.
  </Step>
</Steps>

<Callout kind="success">
  After signup, you land on the dashboard. Bookmark `https://dashboard.mosambee.com` for quick access.
</Callout>

## Initial Dashboard Setup

Customize your dashboard for efficient workforce oversight.

<Tabs>
  <Tab title="Web Dashboard" icon="monitor">
    1. Click **Settings** in the sidebar.
    2. Set your company timezone and working hours.
    3. Enable notifications for attendance alerts.

    ```javascript
    // Example: Update settings via API
    const response = await fetch('https://api.mosambee.com/v1/settings', {
      method: 'PATCH',
      headers: {
        'Authorization': `Bearer ${YOUR_API_KEY}`,
        'Content-Type': 'application/json'
      },
      body: JSON.stringify({
        timezone: 'Asia/Kolkata',
        workingHours: '09:00-18:00'
      })
    });
    ```
  </Tab>
  <Tab title="Mobile App" icon="smartphone">
    Download the Mosambee app from App Store or Google Play.
    1. Log in with your credentials.
    2. Grant location permissions for geo-fencing attendance.
  </Tab>
</Tabs>

## Add Your First Employees

Invite employees to track attendance and performance.

<Steps>
  <Step title="Navigate to Employees" icon="users">
    Go to **Employees > Add New** in the dashboard.
  </Step>
  <Step title="Enter Details" icon="edit-3">
    Fill in name, email, role, and department. Assign a unique employee ID.
  </Step>
  <Step title="Send Invitation" icon="send">
    Click **Invite**. Employees receive an email to set up their profiles and download the app.
  </Step>
</Steps>

For bulk import via API:

<CodeGroup tabs="JavaScript,Python">
  ```javascript
  const employees = [
    { name: 'John Doe', email: 'john@company.com', role: 'Developer' },
    { name: 'Jane Smith', email: 'jane@company.com', role: 'Manager' }
  ];

  const response = await fetch('https://api.mosambee.com/v1/employees/bulk', {
    method: 'POST',
    headers: {
      'Authorization': `Bearer ${YOUR_API_KEY}`,
      'Content-Type': 'application/json'
    },
    body: JSON.stringify(employees)
  });
  ```
  ```python
  import requests

  employees = [
      {'name': 'John Doe', 'email': 'john@company.com', 'role': 'Developer'},
      {'name': 'Jane Smith', 'email': 'jane@company.com', 'role': 'Manager'}
  ]

  response = requests.post(
      'https://api.mosambee.com/v1/employees/bulk',
      headers={
          'Authorization': f'Bearer {YOUR_API_KEY}',
          'Content-Type': 'application/json'
      },
      json=employees
  )
  ```
</CodeGroup>

<ParamField path="employees" param-type="array" required="true">
  Array of employee objects with `name`, `email`, and `role`.
</ParamField>

## Configure Basic Attendance Settings

Set up attendance policies to automate tracking.

<Steps>
  <Step title="Access Attendance Settings" icon="clock">
    Navigate to **Attendance > Settings**.
  </Step>
  <Step title="Define Policies" icon="settings">
    Set check-in/out times, grace periods, and overtime rules. Enable GPS for location-based punches.
  </Step>
  <Step title="Test Configuration" icon="play">
    Have an employee test punch-in via the app to verify setup.
  </Step>
</Steps>
